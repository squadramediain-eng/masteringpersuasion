CThe motion engine — extracted constants

Lifted verbatim from the animator's Remotion source (timing.js, FrameStage.jsx, SvgFrame.jsx) now in reference/ballast-water/remotion/. These are the real numbers — not a guess.

Cue → frame binding

Each scene's settled timecode t ("m:ss") is where it is fully composed. Adjacent scenes split at the midpoint between their settled times; scene 1 pins to frame 0 (cold open). Content beats keep their absolute VO timecode, expressed as a local frame offset.

sec(s) = round(s * 30)   // 30fps
startS[i] = i==0 ? 0 : (settle[i-1]+settle[i])/2
beat.localF = sec(beat.t − startS[i])

The house spring — a buoy

Every entrance rides one spring. No other curve in the film.

BUOY = { damping:200, stiffness:120, mass:0.9 }

Press physics elsewhere depress like a buoy; colour never flips on interaction.

Ghost placeholders

Every content slot is reserved at local frame 0 by a dim clone (auto-generated if the art ships none). It holds the space, then fades out just as its beat lands — so the screen is never blank while it waits.

opacity: interpolate(frame,
  [beatF−6, beatF], [0.16, 0])

The continuous world

<World/> is mounted once for the whole module and never remounts — wave band + ambient sea life persist across every scene. Only cheap transform loops.

drift = Math.sin(frame / 40) * 4   // px sway

FrameStage.World · Root.jsx (World above the scene switch)

Three layers per scene

WORLD continuity glue (persistent) · BASE scaffold + ghosts at t=0 · CONTENT the elements that enter, each on its VO cue. Establish frames fill in 2–3s; build frames arrive one beat per word.

DThe layer-group contract — how flat art becomes a build

The single most valuable upgrade to source art: label the SVG's top-level groups and the renderer builds it beat-by-beat automatically. Unlabelled ("flat") frames still settle in whole over the world — never empty, on-house — but can't build. Our Mastering Persuasion frames already ship this contract (data-layer + data-cue/-relt/-enter).

Labelled top-level groups

<g data-layer="world">…</g>         → World, never remounts
<g data-layer="base">…</g>          scaffold at t=0
<g data-layer="ghost"  data-beat="3">
<g data-layer="content" data-beat="1"
       data-enter="pop" data-cx="…" data-cy="…">

data-beat matches the 1-indexed position in that scene's content[]; its frame comes from the timeline.

The hard positioning rule

Content groups keep their own position. The renderer captures the base transform once and composes the entrance after it — never a CSS transform, which would snap the element to the frame origin.

setAttribute("transform",
   base + " " + entrance)

Centre-scale reads data-cx/cy (the disc centre) so "pop" grows from the middle.

ETweak vocabulary — art-directable without code

Any sub-element tagged data-anim animates every frame; nothing moves unless tagged. Binding data-tweak="name" surfaces it as a live slider in Remotion Studio.

data-anim keywords

spin  data-pivot="cx cy"  [data-speed=deg/frame]
shake data-pivot="cx cy"  [data-amp=deg data-freq]
sway  data-pivot="cx cy"  [data-amp=deg data-freq]
blink                    [data-period=frames data-min]
